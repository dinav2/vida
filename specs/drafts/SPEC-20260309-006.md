## Problem Statement

vida is a command palette with no commands. The app launcher and AI fallback cover search and lookup, but there is no way to trigger discrete actions â€” lock the screen, kill a window, reload Hyprland, run a shell snippet â€” without leaving the palette. This makes vida a launcher, not a command palette.

## Solution

Add a command mode triggered by typing `:` as the first character. Command mode shows a filtered list of named commands. Selecting one runs it. Output either dismisses the palette silently (fire-and-forget) or streams into the result area (palette output). Commands are split into three tiers: built-in system commands, user-defined shell commands in config.toml, and AI commands that reuse the existing streaming infrastructure with a fixed system prompt.

## Scope

**In scope:**
- `:` prefix switches palette to command mode; normal input resumes app/calc/AI routing
- Command list filtered by fuzzy name/description match as user types after `:`
- Built-in commands: `lock`, `sleep`, `reboot`, `shutdown`, `reload-hypr`, `kill-window`, `reload-vida`
- User-defined commands in `config.toml` with `exec`, `output = "none"|"palette"`, `{input}` placeholder
- AI commands: `translate`, `explain`, `define`, `fix`, `summarize` â€” reuse existing AI streaming
- `output = "none"`: exec runs, palette closes, no result shown
- `output = "palette"`: stdout captured and displayed in result area
- `Ctrl+C` on a palette result copies it to clipboard
- Commands searchable by name and description
- Command rows show name + description + optional icon

**Out of scope:**
- Multi-line input / expanded view (SPEC-007)
- Note creation (SPEC-007)
- Extension/plugin system
- Command output longer than ~20 lines (truncated)

## Config Schema

```toml
# Built-in override example
[commands.builtins.lock]
exec = "swaylock -f"   # override default loginctl lock-session

# User-defined commands
[[commands.user]]
name    = "note"
desc    = "Open today's daily note"
icon    = "text-editor"
exec    = "ghostty -e nvim ~/Notes/$(date +%Y-%m-%d).md"
output  = "none"

[[commands.user]]
name    = "todo"
desc    = "Append a task to inbox"
icon    = "list-add"
exec    = "echo '- {input}' >> ~/Notes/todo.md"
output  = "none"

[[commands.user]]
name    = "dict"
desc    = "Look up a word definition"
icon    = "accessories-dictionary"
exec    = "dict {input} 2>&1 | head -20"
output  = "palette"
```
