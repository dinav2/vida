# PRD: App Launching

**ID:** SPEC-20260307-004
**Stage:** DRAFT
**Date:** 2026-03-07
**Priority:** P1

---

## Overview

Currently vida shows a list of matching apps but does nothing when you select one.
Pressing Enter or clicking a row should launch the app and hide the launcher.

---

## Goals

- Select an app with keyboard (arrow keys) or mouse (click) and launch it
- Launcher hides immediately after launching
- Works with standard `.desktop` files (Exec field)

---

## User Flow

1. User opens vida (Super+Space)
2. Types partial app name — results appear
3. Presses ↑/↓ to move selection, or clicks a row
4. Presses Enter (or clicks) — app launches, launcher hides
5. If no row is selected and Enter is pressed — launch first result

---

## Acceptance Criteria

- AC-1: Typing "fire" and pressing Enter launches Firefox (or top match)
- AC-2: Arrow keys move a visible highlight between rows
- AC-3: App launches as a detached process (launcher doesn't block on it)
- AC-4: Launcher hides after launch
- AC-5: %u/%f/%U/%F placeholders in Exec field are stripped (not passed literally)
- AC-6: Clicking a row launches it (same as Enter)
